# Raccourcis Clavier

Cloak DB est conçu pour les développeurs qui préfèrent garder les mains sur le clavier. Chaque action a un raccourci.

Appuyez sur `?` n'importe où dans le studio pour voir la fenêtre des raccourcis.

## Raccourcis Globaux

Ces raccourcis fonctionnent partout dans le studio.

| Raccourci | Action |
|-----------|--------|
| `Cmd+E` | Ouvrir la palette de commandes (recherche de tables) |
| `Cmd+J` | Nouvel onglet requête |
| `W` | Fermer l'onglet actuel |
| `?` | Afficher l'aide des raccourcis clavier |
| `Esc` | Effacer la sélection / Fermer les modales |

## Palette de Commandes

La palette de commandes (`Cmd+E`) permet de rechercher rapidement parmi toutes vos tables.

| Raccourci | Action |
|-----------|--------|
| `↑` / `↓` | Naviguer dans les résultats |
| `Entrée` | Ouvrir la table sélectionnée |
| `Esc` | Fermer la palette |

**Astuces :**
- La recherche inclut les noms de schémas (ex. "public.users")
- Affiche le nombre de lignes pour un contexte rapide
- Gère les fautes de frappe grâce à la recherche floue

## Navigateur de Données

Lors de la visualisation des données d'une table :

| Raccourci | Action |
|-----------|--------|
| `Cmd+S` | Enregistrer les modifications en attente |
| `Cmd+N` | Ajouter une nouvelle ligne |
| `Suppr` | Supprimer les lignes sélectionnées |

## Filtrage

Filtrage rapide piloté au clavier :

| Raccourci | Action |
|-----------|--------|
| `Cmd+F` | Ouvrir le popover de filtre |
| `Cmd+1-9` | Sélectionner un filtre par numéro |
| `m` | Modifier le filtre sélectionné |
| `Suppr` | Supprimer le filtre sélectionné |
| `←` / `→` | Naviguer entre les filtres |
| `Esc` | Désélectionner le filtre |

**Popover de filtre :**
| Raccourci | Action |
|-----------|--------|
| `↑` / `↓` | Naviguer dans les suggestions de colonnes |
| `Entrée` | Sélectionner la colonne / Appliquer le filtre |
| `Retour` | Revenir à la sélection de colonne |
| `Esc` | Fermer le popover |

## Sélection de Cellules

Sélectionnez et modifiez plusieurs cellules à la fois :

| Raccourci | Action |
|-----------|--------|
| `Clic` | Modifier une cellule |
| `Cmd+Clic` | Basculer la sélection de cellule |
| `Shift+Clic` | Sélectionner une plage de cellules |
| `Opt+Clic` | Copier la valeur de la cellule dans le presse-papiers |
| `Cmd+C` | Copier les cellules sélectionnées |

**Workflow d'édition multi-cellules :**
1. `Cmd+Clic` pour sélectionner plusieurs cellules
2. Tapez votre nouvelle valeur
3. Appuyez sur `Entrée` — toutes les cellules sélectionnées se mettent à jour
4. `Cmd+S` pour enregistrer les modifications

## Édition de Cellule

Lors de l'édition d'une cellule :

| Raccourci | Action |
|-----------|--------|
| `Entrée` | Confirmer l'édition |
| `Escape` | Annuler l'édition |
| `Tab` | Confirmer et passer à la cellule suivante |

## Opérations sur les Lignes

| Raccourci | Action |
|-----------|--------|
| `Clic case à cocher` | Basculer la sélection de ligne |
| `Shift+Case à cocher` | Sélectionner une plage de lignes |
| `Clic droit` | Ouvrir le menu contextuel |

## Éditeur de Requêtes

Lors de l'écriture de SQL :

| Raccourci | Action |
|-----------|--------|
| `Cmd+Entrée` | Exécuter la requête |
| `Esc` | Quitter l'éditeur |

## Gestion des Onglets

| Raccourci | Action |
|-----------|--------|
| `Tab` | Naviguer vers l'onglet suivant (boucle) |
| `Shift+Tab` | Naviguer vers l'onglet précédent (boucle) |
| `W` | Fermer l'onglet actuel |
| `Clic molette` | Fermer l'onglet cliqué |

## Notes sur les Plateformes

- **Mac** : Utilise `Cmd` et `Opt`
- **Windows/Linux** : Utilise `Ctrl` au lieu de `Cmd`, `Alt` au lieu de `Opt`

La fenêtre des raccourcis (`?`) affiche automatiquement les bonnes touches pour votre plateforme.
